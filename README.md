# Alex Golowyn Business Website

Business website for Alex Golowyn.

## Nuxt v4 project important details

Project is currently being developed as a SPA using Nuxt v4.

## Setup

```bash
    cd nuxt-app && yarn
```

## Dev

```bash
    cd nuxt-app && yarn dev
```

## Build

```bash
    cd nuxt-app && yarn build
```
